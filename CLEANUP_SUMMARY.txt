================================================================================
                              QUICK START
================================================================================

1. SETUP
   $ cd /path/to/ussop-project
   $ cp .env.example .env
   $ python -m venv venv
   $ source venv/bin/activate
   $ pip install -r requirements.txt

2. DEPLOY WITH DOCKER
   $ docker-compose -f docker/docker-compose.yml up -d
   $ open http://localhost:8080

   OR DEPLOY MANUALLY
   $ cd ussop
   $ python setup.py
   $ python run.py
   $ open http://localhost:8080

3. RUN TESTS
   $ cd ussop
   $ python run_tests.py
